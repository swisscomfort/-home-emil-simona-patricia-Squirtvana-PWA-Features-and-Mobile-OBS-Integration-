# Privacy Policy for Squirtvana PWA

**Last Updated:** November 2, 2025

## 1. Introduction

Squirtvana ("App", "we", "us", or "our") operates the Squirtvana PWA mobile application. This page informs you of our policies regarding the collection, use, and disclosure of personal data when you use our Service and the choices you have associated with that data.

## 2. Information Collection and Use

### 2.1 Automatically Collected Information

Our Service does NOT automatically collect personal information. The App may collect:

- **Device Information:** Device model, OS version, app version
- **Usage Analytics:** Feature usage patterns (optional, if enabled)
- **Crash Reports:** Error logs for debugging (optional)

### 2.2 User-Provided Information

You may provide:
- **API Keys:** OpenRouter, ElevenLabs (stored locally only)
- **Configuration Data:** OBS settings, preferences
- **Generated Content:** Text and audio created through the app

### 2.3 Third-Party Services

The App integrates with:

#### OpenRouter (AI Content Generation)
- **Data Shared:** Your prompts/text input
- **Purpose:** AI content generation
- **Privacy Policy:** https://openrouter.ai/privacy

#### ElevenLabs (Text-to-Speech)
- **Data Shared:** Generated text for voice synthesis
- **Purpose:** Audio generation
- **Privacy Policy:** https://elevenlabs.io/privacy

#### Telegram Bot API (Optional)
- **Data Shared:** Stream notifications, user ID
- **Purpose:** Notifications
- **Privacy Policy:** https://telegram.org/privacy

#### OBS (Local Only)
- **Data Shared:** Scene names, connection info (local only)
- **Purpose:** Stream control
- **Privacy Policy:** N/A (local software)

## 3. Data Storage

### 3.1 Local Storage
- API keys stored in `.env` file (device only)
- Configuration data stored locally
- No data transmitted to our servers
- User has full control of their data

### 3.2 Server Storage
Squirtvana does NOT maintain central servers for:
- User data
- Content history
- API keys
- Configuration backups

### 3.3 Backup
If you backup your device:
- Backup may include app configuration
- Your API keys may be included
- Ensure backup security

## 4. Data Security

We implement security measures including:

- ✅ **HTTPS/SSL:** All external connections encrypted
- ✅ **Local Encryption:** Local storage protected by device security
- ✅ **No Cloud Sync:** Data never leaves your device
- ✅ **No Tracking:** No user tracking or analytics
- ✅ **No Ads:** No advertising or third-party tracking

### 4.1 What We Do NOT Do
- ❌ We do NOT collect personal data
- ❌ We do NOT track user behavior
- ❌ We do NOT sell user data
- ❌ We do NOT run ads
- ❌ We do NOT require login
- ❌ We do NOT maintain user accounts

## 5. User Rights

You have the right to:

1. **Access:** Review what data is stored locally
2. **Delete:** Remove data from your device anytime
3. **Portability:** Export your configuration
4. **Withdraw Consent:** Disable third-party services
5. **Opt-Out:** Use app without sharing data with third parties

## 6. Children's Privacy

Squirtvana does NOT knowingly collect information from children under 13. If you believe we have collected such information, please contact us immediately.

## 7. Changes to This Privacy Policy

We may update this policy occasionally. We will notify you of any changes by posting the new policy on this page and updating the "Last Updated" date.
